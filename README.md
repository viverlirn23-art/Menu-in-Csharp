# Console Array Analyzer (C#)

A lightweight and interactive C# console application designed to process arrays of integers. It demonstrates clean code principles, modular structure (SRP), and safe user input validation.

## Features
- **Dynamic Array Creation:** User defines the size and inputs elements.
- **Data Analysis:** Calculates total sum, average, minimum, and maximum values.
- **Search Functionality:** Checks if a specific number exists in the array.
- **Crash-Proof Input:** Uses `int.TryParse` to prevent exceptions from invalid user inputs (letters, empty strings).

## Tech Stack
- **Language:** C#
- **Platform:** .NET 8.0 (Console Application)
